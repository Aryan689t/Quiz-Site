# 🎯 Quiz Web App

A simple and interactive quiz application built using **HTML, CSS, and JavaScript**.
Users can attempt multiple-choice questions, receive instant feedback, and view their final score.

---

## 🚀 Features

* 📌 Dynamic question loading using JavaScript
* 🎮 Interactive option selection with highlight effect
* ✅ Real-time answer validation
* 📊 Score tracking system
* 🔄 Navigation between questions
* 🎨 Clean and responsive UI

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling
* Arrays & Objects
* State Management (score, current question)
* Basic UI/UX design.

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
quiz-app/

```

---

## ⚙️ How to Run

1. Clone the repository
2. Open `index.html` in your browser
